# OIBSIP - Oasis Infobyte Python Internship Projects

This repository contains the Python projects completed during my Python Programming Internship at Oasis Infobyte. Each project demonstrates different programming concepts, including GUI development, networking, automation, API integration, and utility applications.

---

## Projects

### BMI Calculator
A Python application that calculates Body Mass Index (BMI) based on user-provided height and weight.

**Concepts**
- User Input
- Mathematical Calculations
- Conditional Statements

---

### Client-Server Application
A basic socket programming project demonstrating communication between a client and server.

**Concepts**
- Python Socket Programming
- Networking
- Client-Server Communication

---

### Password Generator
A secure password generator capable of generating strong random passwords.

**Concepts**
- Random Module
- String Manipulation
- Security Fundamentals

---

### Voice Assistant
A voice-controlled assistant capable of performing basic commands using speech recognition.

**Concepts**
- Speech Recognition
- Text-to-Speech
- Automation

---

### Weather Application
A weather application that retrieves current weather information using an online weather API.

**Concepts**
- API Integration
- JSON Parsing
- HTTP Requests
